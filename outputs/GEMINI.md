# Output & Visualization Standards - EduDisruption-Index

## 📊 Deliverables
- **Maps:** Always include a legend, scale bar, and north arrow. Use color-blind friendly palettes (e.g., `viridis`, `magma`).
- **Dashboards:** Use interactive HTML files (e.g., via `folium`) for exploratory analysis.
- **Charts:** Ensure axis labels are clear and units are specified (e.g., "Conflict Fatalities per 100k").

## 🏷 Naming Conventions
- Maps: `YYYYMMDD_mali_edi_risk_vX.png`
- Reports: `YYYYMMDD_analysis_summary.pdf`

## 🎯 Target Audience
- Primary: Education Bridge Initiative (EBI) non-technical field staff.
- Goal: Immediate situational awareness and prioritization support.
